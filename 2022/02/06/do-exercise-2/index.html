CSS 题目总结

div 元素默认的属性

没有任何设置的情况下，div 元素宽高等数值属性默认都为 0，只有两个特殊的属性：

  1. border-top-width:设置一个元素的顶部边框的宽度，默认值为medium (thin—细，medium—中等，thick—粗)。
  2. outline-width：指定轮廓的宽度，位于边框边缘外围。默认值为medium (thin—细，medium—中等，thick—粗)。

背景定位属性 background-position

用处：配合 background-image 属性一起使用，用于设置背景图片在盒子中的位置

该属性取值有三种：xpx ypx（具体像素值） | x% y% | Xpos Ypos（left/right/top/bottom/center）

  • 如果只写第一个水平方向的参数，第二个垂直方向的参数会默认为：center|50%|容器高度的一半px
  • Xpos：规定水平方向的对齐方式,值有left,right,center；Ypos同理
  • x%:规定图片水平方向的距离。这里的x%指的是父级容器的宽度减去图片的宽度的差值的x%

选择器优先级

在不考虑多个选择器优先级叠加的情况下，优先级如下：

!important > 行内 style > id 选择器 > class 选择器 = 伪类 = 属性选择器 > 标签名称选择器 = 伪元素 > 通用选择器（比如*） > 所有的继承样式

所有继承样式的优先级都最低，继承样式内部还会按照这个顺序排列
样式表的来源不同时，优先级顺序为：内联样式 > 内部样式 > 外部样式 > 浏览器用户自定义样式 > 浏览器默认样式。

浮动元素重叠问题

  1. 行内元素与浮动元素发生重叠，边框、背景、内容都会显示在浮动元素之上
  2. 块级元素与浮动元素发生重叠，边框、背景会显示在浮动元素之下，内容会显示在浮动元素之上（有时候也会在浮动元素边上，表现出来即文本脱离了原先的容器）
  3. 以两个块级元素，其中一个左浮动为例
    • 块级元素右边的元素浮动，那么浮动的元素将块级元素挤到其下方；
    • 块级元素的左边元素浮动，就会发生上面的重叠
  4. 以一个行内或行内块，另一个是左浮动的块级元素为例
    • 行内元素右边的元素浮动，那么浮动的元素将不会覆盖它，而是将其挤往左方
    • 行内元素的左边元素浮动，会将其挤到右边；
      即行内元素一定不会和浮动元素发生任何重叠

相邻的两个 inline-block 节点出现间隔

真正意义上的 inline-block 水平呈现的元素间，换行显示或空格分隔的情况下会有间距

出现原因是元素之间的空白符（空格、回车换行等）都会被浏览器处理，HTML 代码中的回车换行被转成一个空白符，在字体不为 0 的情况下，空白符占据一定宽度。

因此解决方法有两种思路：将中间空白消去或中间空白字体大小为 0，还有就是设置字符间距。
